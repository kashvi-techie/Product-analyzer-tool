2. Grid Layout & Spacing

  • Mobile grid: 4 columns, 16px margins, 12px gutters.
  • Spacing scale: 4 / 8 / 12 / 16 / 24 / 32 / 48px.
  • Primary sections: 24px horizontal padding.
  • Bottom CTA safe area: 24px + device inset.

3. Typography System

  • Font: Inter or SF Pro for clinical clarity.
  • H1: 32px / 38px, 800 weight, -0.03em.
  • Body: 17px / 25px, 400 weight.
  • Chips: 13px / 16px, 800 weight, uppercase optional.

4. Key Element Architecture

  • Status ring: 260-320px, centered, 12-16px stroke.
  • Ring label: max two words, state color applied.
  • Input drawer: 88-92% width, 28px radius, bottom anchored.
  • Drawer controls: sliders, chips, decimal fields, clear Save CTA.

Source direction for presentation visuals: Natural Cycles Today, Add Data, Graph, Messages, Cycle Insights, Trends, and Learn surfaces; wording follows the app-tour/help text supplied for this presentation.

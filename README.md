# Product Analyzer Tool

Interactive product-research walkthroughs for cycle and pregnancy care product analysis.

## Included Walkthroughs

- `flo-video-walkthrough/` - Flo product-research walkthrough
- `clue-video-walkthrough/` - Clue product-research walkthrough

The root `index.html` opens the Flo walkthrough by default. The in-page app switcher lets you move between Flo and Clue.

## Deploy

This is a static site.

- Build command: none
- Publish directory: `.`

For GitHub Pages, deploy from the repository root.
